# Basic JS/TS Starter

Quick starter project to run JavaScript and TypeScript examples.

Commands:

- Install dependencies: `npm install`
- Run JS example: `npm run start:js`
- Run TS example (via ts-node): `npm run start:ts`
- Build TS to `dist/`: `npm run build`
